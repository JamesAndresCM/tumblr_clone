## Tumblr Clone

### Descripción 
- Pequeño clone de tumblr. Creado principalmente para el uso de helper en vez de variables de instancia en el controller.

### Características
- Sistema de post con tags y comentarios

### Instalación
```bash
bundle install
rails db:create
rails db:migrate
rails db:seed
rails server
```
